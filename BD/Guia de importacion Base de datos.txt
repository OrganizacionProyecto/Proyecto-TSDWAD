Guia de import para la base de datos. (Basado en Workbench)
1. Crear la base de datos con el nombre "aymara_api_rest".
2. ir a Server > Data Import.
3. Selecciona Import from Self-Contained File y selecciona el archivo aymara_api_rest.sql de la carpeta .BackEnd/BD
4. Elige la base de datos de destino (la que creaste en el paso anterior).
5. Ir a Import Progress y apretar Start Import.

Ahora tienen la base de datos actualizada con los datos de:
- CustomUser: Se creo un Superuser administrador, las credenciales son
  email: admin@gmail.com password: admin123
- Categoria: Se agrego una categoria.
- Producto: Se agregaron tres productos.
